# ACS 반응 작동 안 함 원인 분석 최종 결과

## 사용자 지적

**"에너지 효율이 낮아도 좋아요. 그게 맞는거 같아요. 다만 에너지 효율이 떨어지면 glyoxylate shunt를 통해 가는 반응이 줄어들고 TCA로 가는 반응이 늘어나서 NADH를 더 많이 만들고 대신 탄소를 잃어서 에너지를 만들어야죠."**

→ 사용자 지적이 **완전히 맞습니다!**

---

## 확인된 사실

### 1. ADK1 반응 존재
- ✅ 신규 모델에 ADK1 존재: `AMP + ATP <=> 2ADP`
- ✅ bounds: [-1000.0, 1000.0] (정상)

### 2. ACS 반응 존재
- ✅ 신규 모델에 ACS 존재: `ac_c + atp_c + coa_c --> accoa_c + amp_c + ppi_c`
- ✅ bounds: [0.0, 1000.0] (정상)

### 3. ACS 강제 활성화 테스트 결과

**ACS를 강제로 활성화했을 때**:
- ✅ ACS 플럭스: 0.1 (작동!)
- ❌ 성장률: 0 (경로 불완전)

**결과 분석**:
- ACS 반응 자체는 작동 가능
- 하지만 경로가 완전하지 않아 성장 불가
- CoA 순환은 일어남 (AADa, AADb 반응으로)

---

## 문제 원인

### 이론적으로는 맞지만...

✅ **이론적으로**: ACS + ADK1 조합으로 작동 가능
- ACS: `ATP → AMP + PPi`
- ADK1: `AMP + ATP → 2ADP`
- 에너지 효율이 낮아도 TCA cycle로 보상 가능

❌ **실제로는**: 경로가 완전하지 않음
- ACS를 강제 활성화하면 플럭스 발생
- 하지만 성장률이 0
- **경로가 완전하지 않음**

### 가능한 원인

1. **CoA 초기화 문제**
   - CoA 합성 경로가 불완전할 수 있음
   - CoA 전구체 (panto__R_e, ncam_e)가 필요할 수 있음
   - 레퍼런스 미디어에는 포함되어 있지만 모델에 Exchange가 없을 수 있음

2. **경로 연결 문제**
   - ACS → Acetyl-CoA 생성
   - 하지만 이후 경로 (TCA, Glyoxylate)가 연결되지 않을 수 있음
   - 필수 반응이 누락되었을 수 있음

3. **에너지 생성 경로 문제**
   - ATP 생성 경로가 불완전할 수 있음
   - Electron transport chain이 작동하지 않을 수 있음

---

## 사용자 지적에 대한 결론

### 사용자 지적이 맞습니다!

✅ **에너지 효율이 낮아도 OK**:
- ACS + ADK1 조합으로 작동 가능
- 에너지 효율이 낮으면 TCA cycle로 보상
- NADH 생성 증가, 탄소 손실 (CO2 방출)

### 하지만 실제 문제는...

❌ **경로 불완전성**:
- ACS 반응 자체는 작동 가능 (강제 활성화 시 플럭스 발생)
- 하지만 경로가 완전하지 않아 성장 불가
- CoA 초기화, 경로 연결, 또는 다른 필수 반응 문제

---

## 해결 방안

### 방안 1: 경로 완전성 확인

1. **CoA 합성 경로 확인**
   - CoA 전구체 공급 (pnto__R_e 등)
   - CoA 합성 반응 확인

2. **경로 연결 확인**
   - ACS → Acetyl-CoA → TCA/Glyoxylate
   - 필수 반응 누락 여부 확인

3. **에너지 생성 경로 확인**
   - ATP 생성 경로 (Electron transport chain)
   - NADH → ATP 전환 경로

### 방안 2: 레퍼런스 모델과 비교

- 레퍼런스 모델은 ACS를 사용하지 않고 ACS_ADP를 사용
- 하지만 사용자 지적대로 ACS도 작동 가능해야 함
- 레퍼런스 모델의 경로 구조 확인 필요

### 방안 3: ACS_ADP 추가 (단기 해결)

- 사용자 지적은 맞지만, 경로 완전성 문제로 ACS가 작동하지 않음
- 단기 해결책: ACS_ADP 추가 (레퍼런스 모델 방식)
- 장기 해결책: ACS 경로 완전성 확보

---

## 요약

1. ✅ 사용자 지적: ACS + ADK1 조합으로 작동 가능 (에너지 효율이 낮아도 OK)
2. ✅ 이론적으로 맞음: ACS + ADK1 → AMP 재생성 → ATP 생성
3. ✅ 에너지 효율이 낮으면: TCA cycle로 보상 (NADH 생성, 탄소 손실)
4. ❌ 실제 문제: 경로 불완전성 (ACS는 작동하지만 성장 불가)
5. 🔍 원인 조사 필요: CoA 초기화, 경로 연결, 필수 반응 누락

**결론**: 사용자의 이론적 지적은 완전히 맞습니다. 하지만 실제로는 경로 불완전성 문제가 있어 ACS가 작동하지 않습니다. 경로 완전성을 확보하면 ACS + ADK1 조합으로 작동 가능할 것입니다.
