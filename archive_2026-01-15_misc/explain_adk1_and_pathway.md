# ADK1과 경로 불완전성 설명

## ADK1 (Adenylate kinase)이란?

**ADK1**은 **Adenylate kinase (아데닐레이트 키나제)** 라는 효소입니다.

### 반응식

```
AMP + ATP <=> 2ADP
```

### 역할

1. **AMP 재생성**: AMP를 ADP로 전환
2. **ATP 재생성 경로**: ADP는 ATP로 전환 가능
3. **에너지 조절**: AMP/ADP/ATP 비율 조절

### 왜 중요한가?

**ACS 반응**:
```
ac_c + atp_c + coa_c --> accoa_c + amp_c + ppi_c
```

- ACS는 ATP 1개를 소모하여 AMP를 생성
- AMP는 직접 ATP로 재생성 불가
- **ADK1이 필요**: `AMP + ATP → 2ADP`
- ADP는 ATP로 재생성 가능

### 사용자 지적이 맞는 이유

사용자: "ACS를 통해 AMP가 만들어져도 ATP + AMP로 ADP + ADP 만들 수 있잖아요"

→ **정확히 ADK1 반응**입니다!
- AMP + ATP → 2ADP (ADK1)
- 2ADP → ATP + ADP (ATP synthase 등)
- 결과적으로 AMP를 재활용하여 ATP 재생성 가능

## 경로 불완전성 (Pathway Incompleteness)이란?

### 제가 말한 것

**"경로 불완전성"** = Pathway가 완전하지 않음

### 의미

1. **반응 누락**: 필요한 반응이 모델에 없음
2. **경로 연결 안 됨**: 반응은 있지만 연결되지 않음
3. **초기화 불가**: 시작 메타볼라이트를 만들 수 없음

### 예시

**완전한 경로**:
```
Acetate → (ACS) → Acetyl-CoA → (CS) → Citrate → ... → 성장
```

**불완전한 경로**:
```
Acetate → (ACS) → Acetyl-CoA → ❌ (CS 반응 없음) → 성장 불가
```

또는

```
Acetate → (ACS) → Acetyl-CoA → (CS) → Citrate → ... 
                                                      ↑
                                            CoA 초기화 불가 (CoA 없음)
```

### 현재 상황

1. ✅ ACS 반응: 있음
2. ✅ ADK1 반응: 있음
3. ❌ 경로 불완전성:
   - ACS를 강제 활성화하면 플럭스 발생 (작동!)
   - 하지만 성장률이 0 (경로가 완전하지 않음)
   - 어딘가에서 경로가 끊김

## 경로안정성 (Pathway Stability)?

### 일반적 의미

**경로안정성**은 보통 다음과 같은 의미입니다:
- 경로가 다양한 조건에서도 안정적으로 작동
- 플럭스 변동이 크지 않음
- 외부 조건 변화에 강건함

### 제가 말한 것과는 다름

제가 말한 **"경로 불완전성"**은:
- 경로가 작동하지 않는 이유를 설명
- 반응이 없거나 연결되지 않음
- 경로가 "끊어져 있음"

**"경로안정성"**이 아니라 **"경로 불완전성"**입니다!

---

## 정리

### ADK1
- **이름**: Adenylate kinase (아데닐레이트 키나제)
- **반응**: `AMP + ATP <=> 2ADP`
- **역할**: AMP를 ADP로 재생성하여 ATP 재활용 경로 제공
- **중요성**: ACS로 생성된 AMP를 재활용하는 데 필요

### 경로 불완전성
- **의미**: 경로가 완전하지 않음 (반응 누락, 연결 안 됨, 초기화 불가)
- **현재 상황**: ACS는 작동하지만 경로가 어딘가에서 끊어져서 성장 불가

---

## 현재 문제

**이론적으로는**:
- ACS + ADK1 조합으로 작동 가능 ✅
- 에너지 효율이 낮아도 TCA cycle로 보상 가능 ✅

**실제로는**:
- ACS를 강제 활성화하면 플럭스 발생 ✅
- 하지만 성장률이 0 ❌
- **경로 불완전성** 문제 (어딘가에서 경로가 끊어짐)

**해결 필요**:
- 경로 어디서 끊어지는지 확인
- 누락된 반응 추가
- 경로 완전성 확보
