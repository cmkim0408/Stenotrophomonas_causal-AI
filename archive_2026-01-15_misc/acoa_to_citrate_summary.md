# Acetyl-CoA → Citrate 경로 분석 결과

## 사용자 질문

**"왜 Acetyl-CoA가 Citrate로 안 가는가? ADK1은 발현이 되어야 한다"**

---

## 핵심 발견

### 1. ACS만 강제 활성화 시

**결과**:
- ✅ ACS: 0.1 (작동!)
- ❌ CS: 0.0 (작동 안 함!)
- ❌ ADK1: 0.0 (작동 안 함!)
- ❌ OAA 생성 반응 플럭스 없음!

**문제**: OAA가 없어서 CS가 작동하지 않음!

### 2. ACS + CS 함께 강제 활성화 시

**결과**:
- ✅ ACS: 0.1 (작동!)
- ✅ CS: 0.1 (작동!)
- ✅ ADK1: 0.114108 (작동!)
- ✅ OAA 생성 경로 활성화됨

**해결**: CS를 강제 활성화하면 OAA 생성 경로가 자동으로 활성화됨!

### 3. OAA 가용성 확인

**결과**:
- ✅ OAA를 생성할 수 있음! (최대 생산량: 500)
- ✅ OAA 생성 반응 존재:
  - MDH2: `mal__L_c + q8_c --> oaa_c + q8h2_c`
  - ASPO1: `asp__L_c + h2o_c + o2_c --> h2o2_c + nh4_c + oaa_c`
  - ASPTA: `akg_c + asp__L_c <=> glu__L_c + oaa_c`

---

## 원인 분석

### 문제점

**OAA 초기화 문제**:
1. OAA 생성 경로는 존재함
2. 하지만 자동으로 활성화되지 않음
3. CS가 작동하려면 OAA가 필요한데, OAA가 없음
4. 결과적으로 CS가 작동하지 않음

### 경로 연결

**완전한 경로**:
```
Acetate → (ACS) → Acetyl-CoA
                ↓
              (CS) ← OAA 필요!
                ↓
              Citrate → TCA/Glyoxylate → ...
```

**현재 상황**:
```
Acetate → (ACS) → Acetyl-CoA
                ↓
              (CS) ← OAA 없음! (초기화 안 됨)
                ↓
              ❌ 작동 안 함
```

### 왜 OAA 생성 경로가 자동으로 활성화되지 않나?

**가능한 원인**:
1. **초기 메타볼라이트 부족**: 
   - MDH2는 Malate가 필요
   - Malate를 만들 경로가 초기화되지 않음
   
2. **경로 순환 문제**:
   - OAA를 만들려면 다른 메타볼라이트 필요
   - 그 메타볼라이트를 만들 경로가 초기화되지 않음

3. **에너지 문제**:
   - OAA 생성에 ATP 등이 필요할 수 있음
   - ATP가 부족하면 OAA 생성 경로가 활성화되지 않음

## 사용자 지적에 대한 답변

### "ADK1은 발현이 되어야 한다"

**확인 결과**:
- ✅ ADK1은 CS와 함께 작동함!
- ✅ ACS + CS 강제 활성화 시 ADK1 플럭스: 0.114108
- ✅ ADK1이 정상 작동함

**하지만**:
- ACS만 강제 활성화하면 ADK1이 작동하지 않음
- CS가 작동하지 않아서 ADK1도 작동하지 않음
- 경로가 연결되지 않음

### "왜 Acetyl-CoA가 Citrate로 안 가는가?"

**원인**: **OAA가 없어서 CS가 작동하지 않음**

**해결책**:
1. OAA 생성 경로 초기화 필요
2. 또는 OAA를 초기에 공급
3. 또는 CS를 강제 활성화 (CS가 활성화되면 OAA 생성 경로도 활성화됨)

---

## 해결 방안

### 방안 1: OAA 생성 경로 초기화

**문제**: OAA 생성 경로가 자동으로 활성화되지 않음

**해결**:
- OAA 생성 경로를 초기화하는 부트스트랩 필요
- 예: Malate, Aspartate 등을 초기에 공급
- 또는 OAA 생성에 필요한 반응을 강제 활성화

### 방안 2: 경로 완전성 확보

**문제**: 경로가 완전하지 않아 초기화 불가

**해결**:
- 누락된 반응 추가
- 경로 연결 확인
- 초기화 경로 확보

### 방안 3: CS 반응 강제 활성화 (임시)

**문제**: CS가 작동하지 않음

**임시 해결**:
- CS를 강제 활성화하면 OAA 생성 경로가 자동으로 활성화됨
- 하지만 이것은 임시 해결책

**장기 해결**:
- 경로 초기화 문제 해결 필요

---

## 요약

1. ✅ **CS 반응**: 존재하고 정상 작동 가능
2. ✅ **ADK1 반응**: CS와 함께 작동함
3. ✅ **OAA 생성 경로**: 존재하고 OAA 생성 가능
4. ❌ **문제**: OAA가 초기화되지 않아 CS가 작동하지 않음
5. ✅ **해결**: OAA 생성 경로 초기화 필요

**결론**: 
- Acetyl-CoA → Citrate 경로 자체는 문제없음
- 하지만 OAA 초기화 문제로 경로가 작동하지 않음
- OAA 생성 경로 초기화 문제를 해결하면 경로가 작동할 것

**사용자 지적**: "ADK1은 발현이 되어야 한다"
- ✅ 맞습니다! CS가 작동하면 ADK1도 작동함
- 하지만 OAA 초기화 문제로 CS가 작동하지 않아 ADK1도 작동하지 않음
